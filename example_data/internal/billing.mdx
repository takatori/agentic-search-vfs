# Billing

この文書は内部向けの請求運用メモです。請求グループまたは内部グループに属する利用者だけが参照できます。PUBLIC プロファイルでは `grep` でも `semantic_search` でも表示されません。

## 請求書エクスポート

締め済み期間の請求書 PDF と構造化エクスポートを生成できます。エクスポートには明細、税額、割引、通貨、社内照合 ID が含まれます。大きなエクスポートは期限付き URL で非同期配布します。

## 入金消込

決済代行会社の入金レポートと内部台帳を照合します。金額、手数料、通貨換算が許容範囲を超えてずれた場合は差異として検出されます。手動再試行は監査ログに記録されます。

## 与信と調整

障害補償や契約条件に基づくクレジットは、理由コードと承認ワークフローにひも付けます。売上認識に影響する調整は管理者承認が必要です。

## 実験での確認

`semantic_search "請求書の消込で差異が出たときの対応" /` はこの文書に意味的に近いですが、PUBLIC プロファイルでは検索結果から除外されます。

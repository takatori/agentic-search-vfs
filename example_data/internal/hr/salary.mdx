# 人事評価と給与改定

この文書は人事と内部管理者向けです。評価ランク、給与改定レンジ、例外承認の考え方を記載します。

## 評価サイクル

半期ごとに目標達成度、行動評価、組織貢献を確認します。評価会議では部門間の基準差を調整し、最終ランクを確定します。

## 給与改定

給与改定レンジは評価ランク、職位、採用市場の水準をもとに決定します。例外的な調整には部門長と人事責任者の承認が必要です。

## 権限確認

給与や評価に関する文書は、PUBLIC や BILLING の利用者には表示されません。`grep -ri "給与" /` と `semantic_search "昇給の基準を知りたい" /` の両方で、権限外に露出しないことを確認します。
